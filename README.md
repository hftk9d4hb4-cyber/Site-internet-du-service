# Site V4 — Service de chirurgie viscérale et digestive du CHU de Nice

Version simple, statique et modulaire, compatible GitHub Pages.

## Mise en ligne

1. Décompresser le ZIP.
2. Téléverser le contenu du dossier à la racine du dépôt GitHub.
3. Conserver ou ajouter à la racine du dépôt la photo d'équipe : `equipe-service.jpg`.
4. Commit directly to `main`.

## Fichiers à modifier pour les mises à jour courantes

- Actualités : `data/actualites.json`
- Essais cliniques : `data/essais-cliniques.json`
- Fiches chirurgiens : `data/chirurgiens.json`
- Coordonnées : `data/contact.json`
- Apparence : `css/styles.css`

## Photos des praticiens

Les fiches praticiens pointent vers :

`assets/photos/nom-du-praticien.jpg`

Si une photo manque, le site affiche automatiquement les initiales du praticien.

## Photo d'équipe

La page d'accueil utilise le fichier :

`equipe-service.jpg`

Ce fichier doit rester à la racine du dépôt GitHub.
