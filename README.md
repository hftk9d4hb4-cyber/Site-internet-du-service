# Site final V3 – Service de chirurgie viscérale et digestive du CHU de Nice

Site statique compatible GitHub Pages.

## Mise en ligne

1. Décompresser le ZIP.
2. Charger tout le contenu à la racine du dépôt GitHub.
3. Publier depuis `main` / `/root` dans GitHub Pages.

## Fichiers à modifier pour administrer le site

- `data/actualites.json` : actualités du service.
- `data/chirurgiens.json` : fiches praticiens, liens Doctolib, liens PubMed.
- `data/essais-cliniques.json` : essais cliniques en cours.
- `assets/photos/` : ajouter les portraits des chirurgiens et la photo d'équipe.

## Photos attendues

La page d'accueil utilise si disponible :

- `assets/photos/equipe-service.jpg`

Chaque fiche praticien peut utiliser :

- `assets/photos/patrick-baque.jpg`
- `assets/photos/damien-massalou.jpg`
- `assets/photos/jean-louis-bernard.jpg`
- etc.

Si une photo manque, le site affiche les initiales du praticien.
